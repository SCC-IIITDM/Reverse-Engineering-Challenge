# Reverse-Engineering-Challenge


## Files
- `rev/diff_model.slx` – Unprotected Simulink model containing the differentiator implementation(This will not ne shared to the particpants).
- `rev/diff_model.slxp` – Protected Simulink model containing the differentiator implementation.
- `rev/trial.slx` – Simulation model used to test the protected differentiator block.

## How to Run

1. Open `rev/trial.slx` in Simulink.
2. Make sure `diff_model.slxp` is present in the same `rev/` folder.
3. Click **Run** in Simulink.
4. Observe the output in the **Scope** block.
